Mixed text, code, and markdown LaTeX

A note Quiver include cells – snippets, code, markdown, LaTeX (through mathjax) or diagrams (sequence diagram, flowchart). You can freely mix different types of cells within a note. You can also set different languages for different cells of code.

An impressive code editor

Notebook programmer must make code editing effortlessly. Quiver bags impressive ACE editor code code in cells, supporting syntax highlighting for more than 120 languages, more than 20 tracks, automatic indentation and sangria, and more.

Markdown support

Quiver lets you write in Markdown with the online format and custom CSS options. A preview window makes Markdown live as you type.

LaTeX support

Quiver mathjax used to write mathematical equations written in LaTeX. Note that mathjax does not provide a complete LaTeX environment. Only macros supported in math mode.

Images, files and links

Drag an image into a text cell and saved locally with the note. Drag a file to a text cell and will become a file link that you can click. URLs text cells are detected automatically.

Instant full – text search

Notes are only useful if you can find them quickly. The full – text search Quiver Kit is based on the search, the same technology used to power Spotlight on your Mac. This is how Quiver can search through thousands of notes in the blink of an eye.

Topics beautiful and fully customizable

Quiver comes with several beautifully designed UI themes, and are fully customizable.

Live Preview

Quiver supports live preview markdown cells and LaTeX.

Presentation Mode

Quiver mode provides a full – screen presentation for your notes. This is ideal for use in the classroom, team meetings, presentations, and review notes on their own.

Tags

Quiver supports labeling. You can assign multiple tags to a note or assign a label to multiple notes at once. Combined with notebooks and linked notes, you have many different ways to organize your notes.

Autosave

Quiver automatically saves all your changes as you work, so you never have to worry about losing changes.

Simple data format JSON

Quiver stores all your data in simple JSON files. These data files reside on your local machine. You can easily export your notes in different formats such as HTML and PDF.

Cloud sync

Quiver allows you to synchronize all your notes on multiple computers via Dropbox, iCloud Drive, Google Drive or other cloud service based on files.

Team Collaboration

is easy to collaborate on Quiver. You can create a share so that everyone on your team can view or edit notes in the notebook notebook. This is a great way to build a shared knowledge base for the team.

Version control

Since Quiver stores all data in simple JSON files, it ‘s easy to put all the library (or a specific portable) under version control. This is another way to collaborate with your team.

Backup and recovery

Quiver allows you to back up your entire library with one click. Restore the library from a backup is just as easy.

Integrations

Quiver will not block. Stores data in a single well documented JSON format. Therefore, it is easy to write scripts to integrate Quiver notes with other tools you use. Common scripts are provided in the documentation site Quiver.

Tutorials and documentation

Quiver comes with integrated tutorials and a complete application online wiki. Anyone can contribute to the wiki and make it better.
